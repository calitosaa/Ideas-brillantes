# Knowledge Base — Patrones de Inteligencia

Este directorio contiene los patrones de inteligencia, comportamiento y capacidades
extraídos del análisis de los mejores sistemas de IA del mundo (fuentes públicas).

## Archivos

- `intelligence_patterns.md` — Patrones de razonamiento y comportamiento de élite
- `capabilities_matrix.md` — Matriz de capacidades de los mejores LLMs
- `safety_guidelines.md` — Guías de seguridad y comportamiento ético
- `response_formatting.md` — Mejores prácticas de formato de respuesta

## Uso

Este conocimiento informa:
1. El sistema prompt maestro en `model/system_prompt_*.md`
2. El training data en `training/dataset/`
3. Las definiciones de agentes en `agents/`
